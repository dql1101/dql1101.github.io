二面

  1. char有多少个字节，多少位，能放下一个汉字么？

    char有2个字节，共16位，在Java中汉字是使用Unicode编码的，一般占2个字节，所以char可以可以放下一个汉字。

  2. try catch finally中finally的执行情况

    在我们选择手动抓取来处理异常的时候，如果使用了finally关键字，那么finally关键字中的代码一定会执行；除非在执行前JVM虚拟机被我们关闭了（使用return System.exit(0)代码），那么finally中的语句将不会执行。

  3. finally应用场景

    finally一般用来关闭一些资源和流对象；比如手写文件复制时，将输入流和输出流关闭。

  4. synchronized应用场景？放在相同方法上面，不同线程能同时执行这些方法吗？

    synchronized一般用来处理多线程同步的问题，相当于给方法加了一把锁，线程只有拿到了这把锁之后，才能执行相应的方法。如果在同一个方法上加了锁，那么这个方法在同一时间只能有一个线程来执行。

  9. String能被继承么？

    String类是被final关键字修饰的类，是不能被继承的。

  10. 说说String，StringBUffer，StringBuilder的区别

    三者都是我们常用的用来处理字符串的类，区别在于

    1. String类型是长度不可变的字符串类型，底层new了一个长度为0的字符数组；而当我们对某个String类型的对象进行拼接的时候，会直接在堆内存中new出一个新的String对象来存储字符串；（有说法是在使用+号进行字符串拼接的时候，其实是生成了一个StringBuffer的对象调用了其append方法然后toStirng成了一个Stirng对象）
    2. StringBuffer是可变字符串(默认长度是16)，通过它的append方法来完成字符串的拼接，并且只会产生一个StringBuffer的对象，而且StringBuffer是线程安全的，它底层的方法都被synchronized修饰了。
    3. StringBuilder也是可变字符串，但是它是线程不安全的，其内部方法都没有做同步。

  11. ArrayList与LinkedList的区别

    ArrayList底层是可变数组，通过索引来获取存储的对象。所以使用ArrayList做查询速度快，增删速度慢；
    LinkedList底层是双向链表，增删数据速度回更快。
